## Description

A digital Wallet built to explore Fintech and PayStack Payment gateway

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Features

- User authentication
- Add fund
- Transfer to another Wallet
- Transfer to Bank Account
- Recurring Recurring Charges
- Subscriptions(monthly) Charges
